# Пульт охраны банка

Веб-приложение для мониторинга активности пропусков в банке. Позволяет отслеживать информацию о посетителях, их времени нахождения в банке и статусе пропусков.

### Как установить

Для работы проекта необходимы следующие переменные окружения, которые нужно разместить в файле `.env` в корневой директории проекта:

* `DB_ENGINE` - движок базы данных (например: `django.db.backends.postgresql_psycopg2`)

* `DB_HOST` - хост базы данных

* `DB_PORT` - порт базы данных

* `DB_NAME` - название базы данных

* `DB_USER` - пользователь базы данных

* `DB_PASSWORD` - пароль пользователя базы данных

* `SECRET_KEY` - секретный ключ Django для безопасности (рекомендуется заменить `REPLACE_ME` на свой ключ)

Пример заполнения файла `.env`:
```
# Database
DB_ENGINE='django.db.backends.postgresql_psycopg2'
DB_HOST='checkpoint.devman.org'
DB_PORT='5434'
DB_NAME='checkpoint'
DB_USER='guard'
DB_PASSWORD='your_password'

# Django
SECRET_KEY='your_secret_key_here'
```

Python3 должен быть уже установлен. 
Затем используйте `pip` (или `pip3`, есть конфликт с Python2) для установки зависимостей:
```
pip install -r requirements.txt
```

Для запуска приложения выполните:
```bash
python main.py
```
